# AGENTS

## 项目概览
- 框架：Next.js App Router + React 19 + TypeScript
- UI：Tailwind CSS 4 + Radix UI
- 入口：app/page.tsx

## 常用脚本
- 开发：pnpm dev
- 构建：pnpm build
- 启动：pnpm start
- 规范：pnpm lint

## 目录结构
- app/：路由与布局
- components/：页面与基础组件
- components/ui/：UI 基础组件
- hooks/：通用 hooks
- lib/：工具方法
- public/：静态资源

## 修改约定
- 复用现有组件与样式体系
- 保持 Tailwind 风格一致
- 不新增未使用的依赖
